# OpenEnds

Панель группировки и операции над OpenEnds на странице редактирования проекта.

Что здесь лежит:

- `06-openends-data.js` — извлечение строк, нормализация текста, построение групп, `DisjointSet`.
- `06-openends-actions.js` — выделение, выбор, bulk grouping, работа с pager и контролами выбора.
- `06-openends-ui.js` — построение панели, rescan, статистика, progress bar, рендер списка кластеров.

Важно:

- Эти модули зависят от общего `state` и базовых настроек из `base`.
- Если добавляется новая операция над группами, сначала стоит решить, это data-layer, action-layer или UI-layer.
