# Common

Общие вспомогательные утилиты без привязки к конкретному сценарию.

Что здесь лежит:

- `dom-utils.js` — DOM-хелперы общего назначения.
- `common-utils.js` — текстовые и timing-утилиты вроде `wait()` и нормализации строк.

Важно:

- Модули из этой папки не должны знать о конкретной странице или бизнес-логике.
- Если утилита начинает зависеть от `state`, DOM-структуры конкретной страницы или feature-флагов, ей уже не место здесь.
