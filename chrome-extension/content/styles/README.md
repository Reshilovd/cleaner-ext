# Styles

Статические CSS-стили для UI расширения.

Что здесь лежит:

- `styles.css` — стили панели OpenEnds, Verify-модалок, подсветок, счётчика ID на вкладке `#manual` и вспомогательных UI-элементов расширения.

Важно:

- Эта папка отвечает только за стили.
- Если логика начинает управлять DOM или состоянием, её нужно переносить в соответствующий feature-модуль.
